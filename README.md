4d-plugin-alias-manager
=======================

Function to create hard link, symbolic link, or Finder alias (a.k.a. bookmark).

System
------

* OS X 10.8+ 4D Cocoa
* OS X 10.7+ 4D Carbon

Commands
--------
```
result:=AM Create link (source;target;type)
```
Constants
---------
* Link Alias 2
* Link Hard 1
* Link Symbolic 0
